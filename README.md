Overview

This project presents a Sales Monitoring Dashboard built in Power BI, designed to analyze revenue performance, geographic distribution of stores, and managerial sales tracking.
The dashboard provides a clear visualization of:

Total revenue
Order volume
Average ticket value
Sales distribution by lot
Monthly revenue trends
Geographic store performance
Sales performance by manager

🎯 Business Objective
The goal of this dashboard is to:
Monitor overall sales performance
Identify high-performing sales lots
Track revenue trends over time
Analyze geographic distribution of sales
Evaluate manager-level performance
This solution supports decision-making through interactive filtering and visual KPIs.

📊 Key Metrics (KPIs)
Total Revenue
Total Orders
Average Ticket Value
Revenue by Lot (%)
Monthly Revenue Trend

Geographic Store Distribution
